Exerccio1 Explorar a separabilidade das classes em 2D

Neste exerccio vamos gerar um conjunto de dados bidimensional com quatro classes distintas. Cada classe ser modelada por uma distribuio normal (gaussiana) com mdia e desvio padro definidos pelo enunciado:

  • Classe0: mdia = [2,3], desvio padro = [0,8,2,5]
  • Classe1: mdia = [5,6], desvio padro = [1,2,1,9]
  • Classe2: mdia = [8,1], desvio padro = [0,9,0,9]
  • Classe3: mdia = [15,4], desvio padro = [0,5,2,0]

Geraremos 100 amostras por classe, totalizando 400 pontos. O objetivo visualizar os pontos em um grfico de disperso (scatter plot) e discutir visualmente se um classificador linear seria capaz de separar todas as classes.

In[1]:
# Exerccio 1  Gerar e visualizar o conjunto 2D
import numpy as np
import matplotlib.pyplot as plt

# Semente para reprodutibilidade
np.random.seed(42)

# Definio das mdias e desvios para cada classe
medias = np.array([[2, 3], [5, 6], [8, 1], [15, 4]], dtype=float)
desvios = np.array([[0.8, 2.5], [1.2, 1.9], [0.9, 0.9], [0.5, 2.0]], dtype=float)

# Gerao das amostras
n_por_classe = 100
X_parts = []
y_parts = []
for i, (mu, sigma) in enumerate(zip(medias, desvios)):
    # cada dimenso tem seu prprio desvio; multiplicamos pelo desvio e somamos a mdia
    pts = np.random.randn(n_por_classe, 2) * sigma + mu
    labels = np.full(n_por_classe, i)
    X_parts.append(pts)
    y_parts.append(labels)

# Concatenar todas as classes
X1 = np.vstack(X_parts)
y1 = np.concatenate(y_parts)

print(f"Formato do conjunto: {X1.shape}, rtulos: {y1.shape}")

# Plotar os pontos
cores = ['red', 'green', 'blue', 'orange']
plt.figure(figsize=(6, 5))
for i in range(4):
    plt.scatter(X1[y1 == i, 0], X1[y1 == i, 1], color=cores[i], label=f"Classe {i}", alpha=0.6)

plt.xlabel('x')
plt.ylabel('x')
plt.title('Exerccio 1: classes gaussianas em 2D')
plt.legend()
plt.grid(True)
plt.show()

Formato do conjunto: (400, 2), rtulos: (400,)

Anlise do Exerccio1

O grfico mostra quatro agrupamentos distintos de pontos. Notase que as trs primeiras classes (0,1 e2) ficam relativamente prximas, com sobreposio parcial entre classes 0 e 1 e entre classes 1 e 2. A classe3 est afastada, direita.

Devido a essa sobreposio entre as classes prximas, um classificador linear (como um perceptron simples) no seria capaz de traar uma nica linha reta que separe todas as classes ao mesmo tempo. Em vez disso, seriam necessrias regies de deciso curvas ou mltiplas fronteiras. Redes neurais com funes de ativao no lineares podem aprender essas fronteiras complexas, ajustandose ao contorno dos agrupamentos.

Exerccio2 No linearidade em dimenses mais altas

Aqui vamos estudar um caso em que os dados so originalmente de cinco dimenses (5D) e no so linearmente separveis. Geraremos dois grupos:

  • Classe A: 500 amostras de uma normal multivariada com mdia (\mu_A = [0, 0, 0, 0, 0]) e matriz de covarincia (\Sigma_A) definida pelo enunciado:

(\Sigma_A = egin{pmatrix} 1{,}0 & 0{,}8 & 0{,}1 & 0{,}0 & 0{,}0
0{,}8 & 1{,}0 & 0{,}3 & 0{,}0 & 0{,}0
0{,}1 & 0{,}3 & 1{,}0 & 0{,}5 & 0{,}0
0{,}0 & 0{,}0 & 0{,}5 & 1{,}0 & 0{,}2
0{,}0 & 0{,}0 & 0{,}0 & 0{,}2 & 1{,}0 \end{pmatrix})

  • Classe B: 500 amostras com mdia (\mu_B = [1{,}5, 1{,}5, 1{,}5, 1{,}5, 1{,}5]) e matriz de covarincia

(\Sigma_B = egin{pmatrix} 1{,}5 & -0{,}7 & 0{,}2 & 0{,}0 & 0{,}0
-0{,}7 & 1{,}5 & 0{,}4 & 0{,}0 & 0{,}0
0{,}2 & 0{,}4 & 1{,}5 & 0{,}6 & 0{,}0
0{,}0 & 0{,}0 & 0{,}6 & 1{,}5 & 0{,}3
0{,}0 & 0{,}0 & 0{,}0 & 0{,}3 & 1{,}5 \end{pmatrix})

Aps gerar os dados, aplicaremos Anlise de Componentes Principais (PCA) para reduzir de 5D para 2D e, assim, visualizar os dados em um scatter plot. Para evitar dependncia de bibliotecas externas, a PCA implementada manualmente: centralizamos os dados, calculamos a matriz de covarincia, extramos os autovalores e autovetores com numpy.linalg.eigh e projetamos nos dois autovetores correspondentes aos maiores autovalores.

# Exerccio 2  Gerar dados 5D e reduzir para 2D com PCA manual
import numpy as np
import matplotlib.pyplot as plt

np.random.seed(42)

# Definir mdias e covarincias
mu_A = np.array([0, 0, 0, 0, 0], dtype=float)
Sigma_A = np.array([[1.0, 0.8, 0.1, 0.0, 0.0],
                    [0.8, 1.0, 0.3, 0.0, 0.0],
                    [0.1, 0.3, 1.0, 0.5, 0.0],
                    [0.0, 0.0, 0.5, 1.0, 0.2],
                    [0.0, 0.0, 0.0, 0.2, 1.0]], dtype=float)

mu_B = np.array([1.5, 1.5, 1.5, 1.5, 1.5], dtype=float)
Sigma_B = np.array([[1.5, -0.7, 0.2, 0.0, 0.0],
                    [-0.7, 1.5, 0.4, 0.0, 0.0],
                    [0.2, 0.4, 1.5, 0.6, 0.0],
                    [0.0, 0.0, 0.6, 1.5, 0.3],
                    [0.0, 0.0, 0.0, 0.3, 1.5]], dtype=float)

# Nmero de amostras por classe
n_amostras = 500

# Gerar amostras
A = np.random.multivariate_normal(mean=mu_A, cov=Sigma_A, size=n_amostras)
B = np.random.multivariate_normal(mean=mu_B, cov=Sigma_B, size=n_amostras)

# Concatenao e rtulos
X = np.vstack([A, B])
y = np.array([0]*n_amostras + [1]*n_amostras)

print(f"Formato dos dados: {X.shape}")

# PCA manual
# 1. Centralizar as variveis
X_c = X - X.mean(axis=0)

# 2. Matriz de covarincia
cov_mat = np.cov(X_c, rowvar=False)

# 3. Autovalores e autovetores
eigvals, eigvecs = np.linalg.eigh(cov_mat)

# 4. Ordenar em ordem decrescente
idx = np.argsort(eigvals)[::-1]
eigvals_sorted = eigvals[idx]
eigvecs_sorted = eigvecs[:, idx]

# 5. Selecionar dois maiores
W = eigvecs_sorted[:, :2]  # matriz 5x2

# 6. Projetar
X_proj = X_c @ W

# Mostrar varincia explicada
var_total = eigvals.sum()
exp_var = eigvals_sorted[:2] / var_total
print(f"Varincia explicada pelos dois componentes: {exp_var}")

# Plotar projeo
plt.figure(figsize=(6, 5))
plt.scatter(X_proj[y == 0, 0], X_proj[y == 0, 1], s=12, alpha=0.6, color='blue', label='Classe A')
plt.scatter(X_proj[y == 1, 0], X_proj[y == 1, 1], s=12, alpha=0.6, color='red', label='Classe B')
plt.xlabel('Componente 1')
plt.ylabel('Componente 2')
plt.title('Exerccio 2: Projeo PCA (manual) de 5D para 2D')
plt.legend()
plt.grid(True)
plt.show()

Formato dos dados: (1000, 5)
Varincia explicada pelos dois componentes: [0.52067092 0.1584545 ]

Anlise do Exerccio2

A projeo em duas dimenses evidencia que as classes A e B apresentam sobreposio significativa. Mesmo reduzidas para 2D, as amostras se intercalam em vrias regies do grfico, sugerindo que a distribuio original em 5D no linearmente separvel. A varincia explicada pelos dois maiores componentes principais mostra quanto da informao total preservada na projeo.

Modelos lineares simples no conseguem separar dados to entrelaados; tcnicas mais sofisticadas como redes neurais com camadas ocultas e funes de ativao no lineares so capazes de aprender fronteiras complexas e capturar dependncias no lineares entre variveis.

Exerccio3 Preparar o conjunto de dados SpaceshipTitanic

Utilizaremos o dataset SpaceshipTitanic, disponvel na plataforma Kaggle, para praticar prprocessamento de dados reais. O conjunto dividido em train.csv (registros de aproximadamente dois teros dos passageiros) e test.csv (restante). A coluna alvo Transported indica se o passageiro foi transportado para outra dimenso (valor booleano).

Principais campos em train.csv

  • PassengerId identificador nico do passageiro (formato gggg_pp, em que gggg identifica o grupo e pp a posio dentro do grupo).
  • HomePlanet planeta de origem do passageiro.
  • CryoSleep se o passageiro optou por animao suspensa durante a viagem.
  • Cabin cabine do passageiro (deck/num/side).
  • Destination planeta de destino.
  • Age idade.
  • VIP se pagou por servio VIP.
  • RoomService, FoodCourt, ShoppingMall, Spa, VRDeck gastos em diferentes amenidades do navio.
  • Name nome completo.
  • Transported varivel alvo (True/False).

Tarefas de prprocessamento

  1. Ler o arquivo train.csv usando pandas.
  2. Explorar dados: exibir as primeiras linhas (head()), e contar valores ausentes por coluna.
  3. Tratar valores ausentes:
    • Para colunas numricas (Age, RoomService, FoodCourt, ShoppingMall, Spa, VRDeck), substituir NaN pela mediana da coluna.
    • Para colunas categricas (HomePlanet, CryoSleep, Cabin, Destination, VIP), substituir NaN por 'Unknown'.
  4. Codificar categorias: aplicar onehot encoding com pandas.get_dummies.
  5. Normalizar atributos numricos: ajustar para mdia 0 e desvio 1, calculando (valor - mdia)/desvio.
  6. Visualizar: plotar histogramas de uma ou duas colunas numricas (por exemplo, Age e FoodCourt) antes e depois da normalizao para perceber o efeito da escala.

O cdigo a seguir executa essas etapas e imprime estatsticas e grficos.

# Exerccio 3  Prprocessar o dataset SpaceshipTitanic (se disponvel)
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

# Tentar ler o arquivo train.csv
try:
    df = pd.read_csv('train.csv')
    print(f"Dataset carregado com sucesso. Formato: {df.shape}")
except FileNotFoundError:
    print('Arquivo train.csv no encontrado. Coloque o arquivo na mesma pasta deste notebook para executar o prprocessamento.')
    df = None

if df is not None:
    # Exibir cabealho
    display(df.head())
    print('
Informaes gerais:')
    print(df.info())

    # Definir colunas numricas e categricas
    col_numericas = ['Age', 'RoomService', 'FoodCourt', 'ShoppingMall', 'Spa', 'VRDeck']
    col_categoricas = ['HomePlanet', 'CryoSleep', 'Cabin', 'Destination', 'VIP']

    # Contar valores ausentes
    print('
Valores ausentes por coluna:')
    print(df.isna().sum()[df.isna().sum() > 0])

    # Plotar histogramas antes da normalizao
    for col in ['Age', 'FoodCourt']:
        if col in df.columns:
            plt.figure(figsize=(5, 3))
            df[col].dropna().hist(bins=40, edgecolor='k')
            plt.title(f'{col} antes do tratamento')
            plt.xlabel(col)
            plt.ylabel('Frequncia')
            plt.show()

    # Copiar dados para processamento
    df_proc = df.copy()

    # Tratar valores ausentes nas numricas (mediana)
    for c in col_numericas:
        if c in df_proc.columns:
            mediana = df_proc[c].median()
            df_proc[c] = df_proc[c].fillna(mediana)

    # Tratar valores ausentes nas categricas (Unknown)
    for c in col_categoricas:
        if c in df_proc.columns:
            df_proc[c] = df_proc[c].fillna('Unknown')

    # One-hot encoding
    df_proc = pd.get_dummies(df_proc, columns=[c for c in col_categoricas if c in df_proc.columns], drop_first=False)

    # Remover identificadores se presentes
    for col in ['PassengerId', 'Name']:
        if col in df_proc.columns:
            df_proc = df_proc.drop(columns=col)

    # Normalizar colunas numricas manualmente: (valor - mdia)/desvio
    for c in col_numericas:
        if c in df_proc.columns:
            media = df_proc[c].mean()
            desvio = df_proc[c].std()
            if desvio != 0:
                df_proc[c] = (df_proc[c] - media) / desvio
            else:
                df_proc[c] = 0

    # Plotar histogramas aps a normalizao
    for col in ['Age', 'FoodCourt']:
        if col in df_proc.columns:
            plt.figure(figsize=(5, 3))
            df_proc[col].hist(bins=40, edgecolor='k')
            plt.title(f'{col} aps normalizao')
            plt.xlabel(col)
            plt.ylabel('Frequncia')
            plt.show()

    print('
Prprocessamento concludo. Formato final:', df_proc.shape)
