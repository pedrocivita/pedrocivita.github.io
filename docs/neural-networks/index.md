## Sobre o Projeto

Este repositório contém uma coleção completa de trabalhos teóricos e práticos explorando conceitos fundamentais e aplicações avançadas em redes neurais artificiais e deep learning. O conteúdo está organizado como um site de documentação profissional construído com MkDocs Material, apresentando implementações, análises e pesquisas em múltiplos domínios de machine learning.

O curso aborda tópicos essenciais incluindo preparação de dados, perceptrons, redes neurais multi-camadas (MLPs), autoencoders variacionais (VAEs) e várias arquiteturas de redes neurais aplicadas a problemas do mundo real.

## Tecnologias Utilizadas

Este projeto utiliza uma stack moderna de machine learning baseada em Python:

- **Python 3.8+** - Linguagem de programação principal
- **NumPy** - Computação numérica e operações com arrays
- **Pandas** - Manipulação e análise de dados
- **Matplotlib** - Visualização de dados e gráficos
- **Jupyter Notebook** - Ambiente de desenvolvimento interativo
- **nbformat** - Manipulação de arquivos de notebook Jupyter
- **yfinance** - Obtenção de dados financeiros (para projetos específicos)
