## Sobre o Projeto

Este projeto documenta meu trabalho no curso de Redes Neurais Artificiais e Deep Learning no Insper, como parte do curso de Engenharia de Computação. O conteúdo inclui exercícios teóricos e práticos explorando conceitos fundamentais e aplicações avançadas em redes neurais artificiais e aprendizado profundo.

O trabalho do curso cobre tópicos essenciais incluindo pré-processamento de dados, perceptrons, perceptrons multi-camadas (MLPs), autoencoders variacionais (VAEs) e várias arquiteturas de redes neurais aplicadas a problemas do mundo real.
